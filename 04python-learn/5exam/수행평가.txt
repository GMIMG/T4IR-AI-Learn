실습 예제
[파이썬과 웹 크롤링]
날씨데이터 크롤링 및 파일 저장
1. 기상청 사이트 http://www.weather.go.kr/weather/observation/currentweather.jsp 에 접속하시오
2. 1의 사이트의 모든 지점, 현재기온,체감온도,습도,풍향을 크롤링하여 weather.csv파일로 저장하시오

[정형 데이터처리 RDB]
네이버 오픈 API를 이용한 워드 클라우드 생성
1. 네이버 블로그 정보를 저장할 다음 테이블을 오라클에 생성하시오.
2. 네이버 open api를 이용하여 여자친구 선물을 블로그에서 검색하고 1의 테이블에 저장하시오
3. 2에서 테이블에 저장한 블로그의 내용중 글자수가 2자 이상인 단어로 워드 클라우드를 생성하시오.

[비정형 데이터처리 MongoDB]
네이버 Open API를 이용한 이미지 검색과 MongoDB에 비정형 데이터 저장
1. 네이버 Open API를 이용하여 “박보검” 이미지를 1000개 검색하시오
2. 1에서 검색한 이미지를 MongoDB 의 test 데이터베이스에 저장하시오
평가기준
- 파이썬으로 문자열 처리와 크롤링을 활용하여 데이터 수집하고 처리하는 능력
- 파이썬을 이용한 정형 Data 처리 능력 
- 파이썬의 다양한 함수 활용 능력
- 데이터 추출과 가공, 집약 능력
- SQL을 이용한 정형 Data 처리 능력 
- 수집한 데이터 DB 와의 연동 및 구현 능력 
- NoSQL을 이용한 비정형 Data 처리능력
